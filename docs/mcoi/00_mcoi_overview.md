# MCOI Overview

Purpose: state the role of MCOI Runtime inside Mullu Platform.
Governance scope: Milestone 0 scaffold boundary.
Dependencies: shared docs in `docs/01_shared_invariants.md` through `docs/05_learning_admission.md`.
Invariants: MCOI Runtime does not redefine shared contracts or absorb MAF Core responsibilities.

## Scope

MCOI Runtime is the computer operating intelligence vertical. It owns computer-operation-specific observation, execution, state capture, and operator-loop surfaces built on the shared foundation.

## Out of scope

- Kernel contract redefinition
- Shared policy semantics changes
- Cross-runtime compatibility bridge implementation
